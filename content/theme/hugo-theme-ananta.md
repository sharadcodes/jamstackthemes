# A minimal, research-oriented Hugo theme
Ananta is a lightweight starter kit tailored for individuals in technical and academic fields. It focuses on readability, performance, and a clean, professional presentation.

## Features
* **Academic Ready:** Dedicated layouts for research papers, projects, and professional experience.
* **Privacy & Speed:** Self-hosted Alegreya fonts (WOFF2) to avoid render-blocking Google Fonts and third-party tracking.
* **Modular Homepage:** Easily configurable section order (About, Experience, Posts, etc.) via `hugo.toml`.
* **Technical Support:** Integrated KaTeX for LaTeX equations and Chroma for high-quality code highlighting.
* **Performance First:** Lazy-loaded images, preconnected assets, and a tiny CSS footprint (~8 KB).
* **Accessible & Responsive:** Semantic HTML structure with a mobile-first design.
